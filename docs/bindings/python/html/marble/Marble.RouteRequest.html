RouteRequest Class Reference

from PyKDE4.marble import *

Inherits: QObject
Namespace: Marble

Detailed Description

Points to be included in a route. An ordered list of GeoDataCoordinates with change notification and Pixmap access, similar to QAbstractItemModel.

addVia (  self,
Marble.GeoDataCoordinates  position
)

Insert a via point. Order will be chosen such that the via point is not before the start or after the destination. Furthermore the distance between neighboring route points is minimized

@note: This does not trigger an update of the route. It becomes "dirty"

@todo: Minimizing the distance might not always be what the user wants
